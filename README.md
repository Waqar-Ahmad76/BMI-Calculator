## ‚öôÔ∏è Setup and Installation

This project uses the Tailwind CLI for compilation.

1.  **Clone the repository:**
    ```bash
    git clone [YOUR-REPO-LINK]
    cd bmi-calculator
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the Tailwind Watch command:**
    ```bash
    npm run start
    ```
    (This command compiles the CSS from `./src/input.css` to `./dist/output.css` and watches for changes.)
4.  Open the `index.html` file in your browser to view the application.
